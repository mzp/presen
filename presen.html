CSS によるアニメーション

  • CSS でアニメーションを指定
    • transition-property: opacity 1s linear;
    • opacity の変化は線形に1秒かけておこなう
  • あとは JavaScript でスタイルの値を変更するだけ！
    • $("#picture").css("opacity", "0")
    • 1秒かけて透明になっていく
